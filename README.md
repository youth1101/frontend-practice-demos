# frontend-practice-demos
前端成长记录仓库，沉淀学习过程中的各类实战 Demo，从基础布局到复杂交互，从原生开发到框架应用，完整还原前端学习的进阶路径，所有代码均可直接运行预览，欢迎 Star & Fork 一起交流学习！

## 📁 仓库结构
- `html-css-demos/`：HTML+CSS 基础布局、动画、响应式实战Demo
- `javascript-demos/`：原生JavaScript 交互、算法、Web API实战Demo
- `vue-demos/`：Vue 框架相关实战Demo
- `react-demos/`：React 框架相关实战Demo

## 🚀 如何运行
1.  克隆仓库到本地：`git clone git@github.com:youth1101/frontend-practice-demos.git`
2.  进入对应Demo文件夹，直接用浏览器打开html文件即可运行预览
3.  框架类Demo会在对应文件夹内补充单独的运行说明
